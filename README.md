# Express JWT authentication using Prisma

You can check the blog post [here](https://dev.to/mihaiandrei97/jwt-authentication-using-prisma-and-express-37nk).

## Setup

```
npm install
npx prisma migrate deploy
```

## Lint

```
npm run lint
```

## Test

```
npm run test
```

## Development

```
npm run dev
```
